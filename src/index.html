Nazywam się Monika Radosiewicz i jestem początkujacym front-end developerem

Pasja

Moja przygoda z web-developmentem zaczęła się około pół roku temu i pochłonęła mnie bez reszty. Programowanie jest jedną z niewielu rzeczy, która potrafi wprowadzić mnie w stan 'flow' i sprawić, że świat wokół przestaje istnieć.

Rozwój

Nieustannie uczę się nowych rzeczy, a ucząc się próbuje zaimplementować je w swoich projektach. Uwielbiam stawiać sobie nowe wyzwania, udowadniając, że nawet najtrudniejsze z pozoru problemy, da się rozwiązać - tylko wymaga to czasu.

Marzenia

Chciałabym nauczyć się tylu rzeczy, jest jeszcze tak wiele do odkrycia w tej branży. Moim celem jest zostanie ekspertem, magikiem, spod ktorego palców płyną cuda.Droga jest długa i wcale niełatwa - ale jestem wystarczająco zmotywowana.

Umiejętności

CSS3

JavaScript

HTML5

Staram się dbać o semantykę i dostępność tworzonych stron, stosując się do obowiązujących standardów HTML5 oraz metodologii BEM. Lubię się bawić CSSem i eksperymentować z jego najnowszymi featurami, zaś dzięki technologiom yakim jak Flexbox, czy CSS Grid, witryny, które tworzę są w pełni responsywne. Dzięki znajomości JavaScriptu, w tym EcmaScript 6, jestem w stanie tchnąć w nie trochę życia i umożliwić interakcję z użytkownikiem.

JQuery

React

Node

Preferuję Vanilla Javascript, dlatego nie korzystam z JQuery zbyt często, natomiast stworzyłam kilka projektów z jego pomocą i poradziłabym sobie w razie potrzeby. Zakochałam się natomiast w Reacie i cały czas szlifuję swoje umiejętności, zgłębiając tajniki zaówno React.jsR jak i Redux.js. Zdarzyło mi się też eksperymentować z Node.js - i choć znam tylko podstawy, mam plany by poszerzyć wiedzę i zająć się również back-ndem tworzonych stron.

SASS

Bootstrap

Materialize CSS

Uwielbiam korzystać z preprocesora SASS - jego funkcjalności, takie jak zagnieżdżanie selektorów, zmienne, mixiny czy dziedziczenie - niesamowicie przyspieszają pracę i usprawniają pisanie kodu. Mimo, że wolę pisać style samodzielnie, czasami zdarza mi się użyć frameworka CSS, jakim jest Bootstrap (w wersji zarówno 3 , jak i 4) lub Materialize CSS.

GIT

Gulp

Webpack

Podczas bieżącej pracy, korzystam z systemu kontroli wersji GITi menadżera pakietów NPM, zaś swoje projekty przechowuję na Githubie i Gitlabie. Do automatyzacji czynności, takich jak kompilacja SASSa do CSSa czy minifikacja skryptów, najczęściej używam Gulpa. Z kolei podczas tworzenia aplikacji w React.js doskonale sprawdza się Webpack, który w odróżnieniu od tusk runnerów, ma nieco większe możliwości i oprócz automatyzacji, oferuje również "bundlowanie" plików.

Projekty

Gra w Memory

Wykonana w czystym JavaScripcie gra w Memory, oferująca różne poziomy trudności. Czas rozgrywki każdego poziomu, zapisywany jest local storage, dzięki czemu użytkonik ma możliwość pobijania swoich rekordów oraz porównywania obecnego wyniku z poprzednim.

  • JavaScript
  • CSS3
  • SASS
  • Local Storage

Gra w Sudoku

Klasyczna gra w Sudoku odzworowana przeze mnie w Reacie, przy pomocy biblioteki Sudoku-Umd oraz Material-UI. Gracz ma możliwość zmiany poziomu trudności rozgrywki, zapisania gry lub jej wczytania, sprawdzenia tablicy lub jej rozwiązania.

  • React.js
  • Materialu-UI
  • Local Storage

Kanban Board

Tablica, pomagająca w zarządzaniu bieżącymi zadaniami i projetkami. Projekt został napisany w obiektowym JavaScripcie, z użyciem biblioteki JQuery i JQuery UI, które umożliwiają przenoszenie elementów na stronie (drag'n'drop). Stan tablicy zapisywany jest w REST- owym API, dzięki czemu wszystkie kolumny i karteczki pozostają na miejscu, nawet po odświeżeniu strony.

  • JavaScript
  • OOP
  • JQuery
  • CSS3

Edytor zdjęć

Prosty edytor zdjęć, który dzięki zasosowaniu technologii CSS variables i HTML5 Canvas, umożliwia nakładanie różnego rodzaju filtrów, przycinanie oraz obracanie obrazka.

  • JavaScript
  • Materialize CSS
  • CSS Variables
  • HTML5 Canvas

Wyszukiwarka użytkowników GitHuba

Jeden z pierwszych projektów wykonanych przeze mnie w Reacie. Wyszukiwarka korzysta z API, udostępnianego przez GitHub i pozwala na wyszukiwanie użytkownikó po loginie.

  • React.js
  • CSS3
